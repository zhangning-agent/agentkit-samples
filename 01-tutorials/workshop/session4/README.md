### 依赖安装

您可以通过 `pip` 工具来安装本项目依赖：

```bash
pip install -r requirements.txt
```

或者使用 `uv` 工具来安装本项目依赖：

```bash
# 使用 `pyproject.toml` 管理依赖
uv sync

# 使用 `requirements.txt` 管理依赖
uv pip install -r requirements.txt
```
